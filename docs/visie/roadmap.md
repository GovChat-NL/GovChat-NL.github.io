# Roadmap

GovChat-NL wordt actief doorontwikkeld door een groeiende community van overheidsorganisaties en ontwikkelaars.

## Actuele roadmap

De meest actuele roadmap is te vinden op de website: **[govchat-nl.nl/roadmap](https://www.govchat-nl.nl/roadmap/)**

## Bijdragen

GovChat-NL is open-source en staat open voor bijdragen van de community:

- **GitHub**: [github.com/GovChat-NL/GovChat-NL](https://github.com/GovChat-NL/GovChat-NL)
- **Issues en feature requests**: via GitHub Issues
- **Pull requests**: welkom voor bugfixes, nieuwe features en documentatie-verbeteringen

## Community

Er vinden regelmatig community-bijeenkomsten plaats om de ontwikkelrichting te bespreken. Neem contact op via [ai@prvlimburg.nl](mailto:ai@prvlimburg.nl) voor meer informatie over deelname.
