# Omgevingsvariabelen

GovChat-NL biedt diverse omgevingsvariabelen om het platform aan te passen aan je organisatie. Hieronder staan de GovChat-NL-specifieke variabelen.

## Welkomstberichten & UI

| Variabele | Standaard | Beschrijving |
|-----------|-----------|-------------|
| `EMPTY_CHAT_WELCOME_MESSAGE` | `"welkom bij GovChat-NL"` | Welkomstbericht in een lege chat |
| `LOGIN_SCREEN_SUBTITLE` | *(geen)* | Ondertitel op het loginscherm |

## Interface-instellingen

| Variabele | Standaard | Beschrijving |
|-----------|-----------|-------------|
| `ENABLE_CONTROLS_BUTTON` | `False` | Toon de controls-knop in de chat |
| `SHOW_CHANGE_PASSWORD` | `False` | Toon de optie om wachtwoord te wijzigen |
| `ALLOW_USERNAME_EDIT` | `False` | Sta gebruikersnaam bewerken toe |
| `SHOW_ABOUT_TAB` | `False` | Toon het 'Over'-tabblad |
| `SHOW_OVERVIEW_IN_DROPDOWN` | `False` | Toon overzicht in dropdown-menu |
| `SHOW_WIDESCREEN_MODE` | `False` | Toon breedbeeldmodus-optie |
| `SHOW_ARCHIVED_CHATS` | `False` | Toon gearchiveerde chats |

## Features

| Variabele | Standaard | Beschrijving |
|-----------|-----------|-------------|
| `ENABLE_CHAT_KNOWLEDGE_ATTACHMENT` | `False` | Kennisbank-bijlagen in chat |
| `ENABLE_CHAT_REFERENCE_CHATS` | `False` | Referentie-chats functie |
